
# Noni Front End Developer Coding Challenge

## OBJECTIVE

To create a WordPress plugin using a starter template that connects to a news API and displays the data.

## INSTRUCTIONS

1.  Create a place in the admin panel where an admin can enter in the token for the API endpoint
2.  Create a shortcode code [noni-worpdress-code] that displays a list of new articles using the provided API
3.  Add the ability to filter the news articles from the API by title from an input box
4. Display an error message if nothing is found matching the search criteria

API Endpoint: [https://newsapi.org/v2/top-headlines?country=us&apiKey=9e56665dd12747d699700625112fe619](https://newsapi.org/v2/top-headlines?country=us&apiKey=9e56665dd12747d699700625112fe619)

Use any technologies that you are comfortable with.

Style your creation as magnificently as you can.

##  Submit
